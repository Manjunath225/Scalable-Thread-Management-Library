Code Examples

Simple, Intuitive API

Clean, expressive API for thread management that makes concurrent programming accessible.

thread_pool_example.cpp
C++
#include <threadsync.h>
#include <iostream>
#include <vector>

int main() {
    // Create thread pool with 8 worker threads
    threadsync::ThreadPool pool(8);
    
    // Vector to store results
    std::vector<std::future<int>> results;
    
    // Submit 100 tasks to the thread pool
    for (int i = 0; i < 100; ++i) {
        results.emplace_back(pool.submit([i] {
            // Simulate computation
            std::this_thread::sleep_for(std::chrono::milliseconds(10));
            return i * i;
        }));
    }
    
    // Collect results
    int total = 0;
    for (auto& result : results) {
        total += result.get();
    }
    
    std::cout << "Total: " << total << std::endl;
    return 0;
}
synchronization_example.cpp
C++
#include <threadsync.h>

// Parallel matrix multiplication using barriers
Matrix parallel_multiply(const Matrix& A, const Matrix& B) {
    int n = A.rows();
    Matrix result(n, n);
    
    // Create barrier for synchronization
    threadsync::Barrier barrier(4);
    
    // Divide work among 4 threads
    threadsync::ThreadPool pool(4);
    
    for (int t = 0; t < 4; ++t) {
        pool.submit([&, t] {
            int start = t * n / 4;
            int end = (t + 1) * n / 4;
            
            // Phase 1: Compute partial results
            for (int i = start; i < end; ++i) {
                for (int j = 0; j < n; ++j) {
                    result(i, j) = 0;
                    for (int k = 0; k < n; ++k) {
                        result(i, j) += A(i, k) * B(k, j);
                    }
                }
            }
            
            // Synchronize before next phase
            barrier.wait();
            
            // Phase 2: Additional processing...
            // ...
        });
    }
    
    pool.wait_for_completion();
    return result;
}

Getting Started

Integrate in Minutes

Follow these simple steps to integrate ThreadSync Pro into your project.

1

Download and Install

Get the latest version for your platform or build from source. Package managers available for all major platforms.

2

Include Headers

Add the ThreadSync Pro include directory to your project and include the main header: #include <threadsync.h>

3

Link Library

Link against the ThreadSync Pro static or dynamic library in your build system (CMake, Make, Visual Studio, etc.).

4

Create Thread Pool

Instantiate a ThreadPool object with your desired number of worker threads and start submitting tasks.
